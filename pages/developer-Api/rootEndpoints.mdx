# Root Endpoint

Here’s a quick rundown of the core API categories we offer. Each section links to a dedicated page with in-depth details on endpoints, request/response formats, and usage examples.

## User Data: Retrieval & Insertion

### Cards & Consents

- **List Installed Cards**
  Retrieve any brand or loyalty “cards” a user has installed.  
  <ApiSection 
  endpoint={{
    method: "POST",
    path: "/api/v1/list-installed-cards",
    description: "Retrieve any brand or loyalty 'cards' a user has installed.",
    queryParams: [
      { name: "phone_number", required: true, placeholder: "Enter phone number" }
    ],
    headers: [
      { name: "token", required: true, placeholder: "Enter token" }
    ]
  }}
  />


- **Request Consent**
  Send a request prompting the user to grant or reject data access.  
  <ApiSection 
  endpoint={{
    method: "POST",
    path: "/api/v1/request-consent",
    description: "Send a request prompting the user to grant or reject data access.",
    requestBody: {
      phone_number: { type: "string", placeholder: "Enter phone number" },
      access_token: { type: "string", placeholder: "Enter access token" },
      card_name: { type: "string", placeholder: "Enter card name" },
      expiry: { type: "string", placeholder: "Enter expiry date" },
      bid_value: { type: "number", placeholder: "Enter bid value" }
    }
  }}
  />

- **Get Consent**
  Retrieves a list of cards for which the user has already granted consent.  
  <ApiSection 
  endpoint={{
    method: "POST",
    path: "/api/v1/list-consented-cards",
    description: "Get a list of consented cards",
    queryParams: [
      { name: "phone_number", required: true, placeholder: "Enter phone number" }
    ],
    headers: [
      { name: "token", required: true, placeholder: "Enter token" }
    ],
    requestBody: {
      brand: { type: "string", placeholder: "Enter brand name" },
      location: { type: "string", placeholder: "Enter location" },
      purchase_category: { type: "string", placeholder: "Enter your purchase category" },
      brand_category: { type: "string", placeholder: "Enter your brand category" },
      Date: { type: "string", placeholder: "Enter date" },
      currency: { type: "number", placeholder: "Enter currency" },
      payment_method: { type: "string", placeholder: "Enter your payment method" },
      total_cost: { type: "number", placeholder: "Enter your total cost" },
      inr_total_cost: { type: "number", placeholder: "Enter inr cost value" },
      usd_total_cost: { type: "number", placeholder: "Enter usd cost value" }
    }
  }}
  />
